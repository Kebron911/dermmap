All features

Everything built for one workflow: dermatology documentation in the exam room.

Ten purpose-built features. Zero unnecessary ones.

Anatomical Body Map

Precise full-body SVG diagram with anterior, posterior, lateral, and head detail views. Tap to place markers on exact lesion locations. Pinch to zoom for precise positioning. Prior visit markers visible in distinct color for easy comparison. New markers pulse to confirm placement, providing instant visual feedback.

  • Multi-view anatomical diagrams
  • Touch-optimized placement
  • Pinch-to-zoom controls
  • Historical marker comparison
  • Pulse confirmation feedback

Clinical Photo Capture

Camera opens instantly when you place a marker. Circular framing guide ensures consistent composition. Auto-capture triggers when the image is stable and in focus, eliminating manual intervention. Support for multiple photos per marker. Photos attach automatically without manual linking required.

  • Instant camera launch
  • Circular framing guide
  • Auto-focus detection
  • Multi-photo support
  • Automatic attachment

Dermoscopy Lens Support

Compatible with industry-standard adapters including DermLite, 3Gen, and Canfield systems. Auto-detects attached lens and automatically adjusts exposure and white balance for optimal image quality. Dermoscopic and clinical photos stored separately with clear labeling for easy reference and comparison.

  • Multi-adapter compatibility
  • Auto-detection system
  • Smart exposure adjustment
  • White balance optimization
  • Organized photo categorization

Structured Documentation

Bottom sheet interface with quick-tap fields for clinical classification. Includes size (1-6mm+ with custom entry), shape, color, border characteristics, symmetry assessment, and action type selection (monitor, biopsy scheduled, biopsy performed, excision, referral). Features sensible defaults to accelerate documentation. Free-text notes field available for provider-only comments and clinical observations.

  • Quick-tap field selection
  • Standardized classifications
  • Custom size entry
  • Action tracking
  • Provider notes field
  • Smart defaults

Offline Mode

Complete offline operation with local AES-256 encryption ensuring all data remains secure even without network connectivity. Mutations are queued automatically and replayed on reconnect with built-in conflict resolution. Offline mode is a first-class feature, not a fallback consideration — exam rooms have unreliable WiFi, and we planned for that.

  • Full offline operation
  • AES-256 encryption
  • Automatic queue management
  • Conflict resolution
  • Seamless reconnection
"Exam rooms have bad WiFi. We planned for that."

Longitudinal Lesion Tracking

Every lesion builds a complete longitudinal history across visits. Access photo carousel showing all images chronologically. Compare images side-by-side across time points. Track structured data changes over time with visual indicators. Integrate biopsy results and pathology notes directly in the lesion record. Color-coded status indicators provide at-a-glance assessment of lesion status.

  • Complete visit history
  • Photo carousel
  • Side-by-side comparison
  • Historical data tracking
  • Biopsy integration
  • Status indicators

PDF Export

One-tap generation of PDF/A-compliant clinical documents suitable for EHR attachment and long-term archival. Documents include patient information, visit date, annotated body map with numbered markers, per-lesion photo grids, structured data summary table, and provider signature line. Typical file size under 5MB. Integrates seamlessly with every major EHR system as a standard document attachment.

  • One-tap PDF generation
  • PDF/A compliance
  • Patient information
  • Annotated body map
  • Photo grids
  • Signature line
  • EHR compatible

Role-Based Access

Three precisely defined roles — Medical Assistant, Physician, and Practice Manager — each with scoped permissions matching their clinical and administrative responsibilities. No shared logins; each user has individual credentials. Session timeout with biometric or PIN resume capability. Break-glass emergency access available with mandatory justification logging for compliance and accountability.

  • Three defined roles
  • Scoped permissions
  • Individual credentials
  • Session timeout
  • Biometric resume
  • Emergency access logging

Audit Log

Immutable, searchable audit log recording every action within the system for compliance and accountability. Logs retained for 6 years meeting regulatory requirements. Exportable to CSV format for integration with compliance workflows. Search by user, date, action type, or affected record for rapid investigation and review.

  • Immutable records
  • Searchable index
  • 6-year retention
  • CSV export
  • Multi-field search
  • Compliance ready

Practice Analytics Dashboard

Comprehensive analytics dashboard providing visibility into practice productivity and adoption metrics. Track visits per day, week, and month. Monitor average documentation time per visit. Analyze photo utilization patterns. View provider adoption metrics and engagement trends. Built-in ROI calculator converts visit volume and medical assistant hourly cost into quantified time savings and operational benefits.

  • Visit metrics
  • Time tracking
  • Photo analytics
  • Adoption metrics
  • ROI calculator
  • Productivity insights
